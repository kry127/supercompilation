### Program example
```
sum (squares (upto 1 n)) 0
 where
  sum     = \xs . \a .
          case xs of
          []     -> a
          x : xs -> sum xs (x + a)
          esac
          
  squares = \xs.
          case xs of
          []     -> []
          x : xs -> (x * x) : squares xs
          esac
          
  upto   = \m. \n.
          case (m > n) of
          True -> []
          False -> m : (upto (m + 1) n)
          esac
```
